## üìù Project Description

**Student Hub** is a comprehensive web interface designed to help students at HUIT (Ho Chi Minh City University of Industry and Trade) efficiently manage their academic life. It provides a central dashboard for organizing courses, assignments, study materials, and personal planning.

[cite\_start]This repository contains the front-end structure (HTML/CSS/JS) for the core pages of the application, focusing on a modern and engaging user experience.

## üé® Design & UI Highlights

The project is built with a modern design aesthetic, featuring:

  * *Dark Theme:** Uses a consistent dark gradient background (`linear-gradient(135deg, #0f0f0f, #1a1a2e, #16213e)`) across the application for a professional and eye-friendly experience.
  * *Glassmorphism/Opacity:** Main components (sidebar, info boxes, course cards) utilize background blurring (`backdrop-filter: blur(10px)`) and opacity to achieve a stylish, layered Glassmorphism effect.
  * *Gradient Typography:** Large titles and key elements employ vibrant color gradients (e.g., `linear-gradient(90deg, #ffffff, #a0a0ff)`) to enhance visual appeal.
  * *Micro-Interactions:** Most cards and buttons feature smooth `transition` and `transform` effects on hover, making the interface dynamic and highly responsive.
  * *Animations:** Components fade into view (`fadeIn` / `fadeInUp`) for a smooth loading experience.

## ‚ú® Key Modules & Pages

The application is structured around several dedicated modules:

| Module Page | Description | Core Functionality |
| :--- | :--- | :--- |
| **Trangchu.html** | Main Dashboard | Displays student overview, quick course links, weekly schedule, and calendar events. |
| **DangKyDangNhap.html** | Authentication | [cite\_start]Features a smooth **Toggle Panel** effect to switch between the Login and Register forms. |
| **Kehoach.html** | Study Plans/Tasks | **3-Column Layout** (Schedule, Main Tasks, Stats) with tasks visually categorized by completion status. |
| **Ghichu.html** | Notes Management | [cite\_start]Allows adding, organizing, and favoriting notes, using color tags for quick classification. |
| **Baitap.html / Tieuluan.html** | Assignments Tracking | Uses a **2x2 Grid** layout to categorize tasks by status: **New**, **Submitted**, **In Progress**, and **Overdue**. |
| **ThongTin.html** | Student Profile | Presents comprehensive academic and personal details, including family relations. |
| **Tailieuhoctap.html** | Resources/Platforms | Lists external platforms categorized by function (Learning, Job Tool, Design, etc.). |
